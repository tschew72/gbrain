---\ntitle: EVVO workbook field coverage\ntype: note\ntags:\n  - m365\n  - workbook\n  - reporting\n---\n\n# EVVO workbook field coverage\n\nThe Excel workbook tabs for the project dashboard currently expose only schedule fields.\n\n- VAPT: status as of, client company name, project name, start date, milestone dates, AM, quotation, alias\n- Consultant: status as of, client company name, project name, start date, estimated end date, AM, quotation, alias\n- CISO: status as of, client company name, project name, start date, estimated end date, AM, quotation, alias\n\nThe workbook does not contain explicit columns for:\n- Pending Tasks\n- Client Requests\n- Unread\n- Unreplied\n\nFor future dashboards, keep those fields as Not listed in workbook unless Outlook or Teams provides a verified update.\n
