## Rule

Do not lump live projects into grouped summaries.
Each project must be listed individually with its latest action, follow-up, or blocker.

## Examples

- HomeTeamNS: report internal audit prep and evidence-pack follow-up.
- Oxpay: report current SOC / phishing / FINTEL follow-up.
- PIL: report ISMS manual / Year 2 audit follow-up.
- SESAMi: report CTM / external-audit follow-up.
- ST Engineering: report latest clarification / risk-plan follow-up.
- Good Bards, Mapletree, Precepts, Valuemax: report the latest active task or review step for each.

## Purpose

This keeps the PM report readable and prevents active projects from being hidden inside generic grouped labels.
